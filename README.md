# hilite

Trivial terminal output colorizer.

## USAGE:

```bash
stuff | hilite --red red_regex -b blue_regex ...
```

Recognized colors:
 * -w, --white,
 * -r, --red,
 * -m, --magenta,
 * -y, --yellow,
 * -g, --green,
 * -c, --cyan,
 * -b, --blue

*Protip*: you can use `less -R` to page colorized output.
